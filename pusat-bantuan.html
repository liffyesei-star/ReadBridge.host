Pertanyaan Umum

Login Google gagal sinkronisasi ke server

Tunggu backend selesai redeploy, lalu coba login ulang. Jika masih gagal, hapus sesi lama dari browser atau gunakan mode incognito untuk memastikan token login baru dibuat.

Pembayaran Midtrans tidak terbuka

Pastikan akun sudah login, buku dipilih dari halaman detail, dan backend Render sudah memiliki konfigurasi Midtrans. Jika tombol kembali menampilkan error konfigurasi, backend perlu redeploy.

Buku belum masuk perpustakaan setelah bayar

Buku masuk setelah status pembayaran sukses diterima dari webhook Midtrans. Jika pembayaran sudah sukses tetapi koleksi belum berubah, cek kembali beberapa menit kemudian.

Data profil tidak berubah di halaman lain

Refresh halaman setelah menyimpan pengaturan. ReadBridge menyimpan beberapa preferensi tampilan di browser, jadi perubahan akan ikut terbaca di tab lain.
